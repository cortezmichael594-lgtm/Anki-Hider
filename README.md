## Hider (by Ankicraft)

Every distraction Anki puts on screen pulls your attention away from the one thing that matters: the card in front of you. Hider strips the interface down to just that card — and puts everything back the moment you leave. 🎯

Hider gives Anki the focus mode it's always been missing.

* 🪟 **Hide the menu bar, top toolbar, and bottom bar** during review — they all reappear automatically when you navigate away, so nothing is lost.
* 📜 **Remove the card scrollbar** for an even cleaner look; the mouse wheel still works normally.
* 🎯 **Hide the Hard and Easy answer buttons** — only Again and Good remain, and their keyboard shortcuts are blocked too so you can't answer by accident.
* 🖱️ **Auto-hide the mouse cursor** after a configurable idle time (1–60 s); it comes back the instant you move the mouse.
* ▶️ **Start reviewing automatically** when you open a deck, skipping the overview screen entirely (only when there are due cards).
* ⚙️ **Every feature can be toggled independently** — enable only what works for your workflow.
* 🌍 **Interface available in 20+ languages**, matching Anki's own language setting.

## HOW TO USE

1. Install the add-on and restart Anki.
2. Open any deck and click Study Now — the configured elements hide automatically.
3. Leave the review screen at any time; everything reappears on its own.
4. To change what gets hidden: **Tools → Add-ons → Hider → Config**.

## SETTINGS

Go to **Tools → Add-ons → Hider → Config** to toggle each feature individually and set the cursor idle timeout (default: 3 s).

## COMPATIBILITY

Tested on Anki 24.06+ (Qt6). Tested primarily on Windows; macOS and Linux should work but please report any issues in the comments.
